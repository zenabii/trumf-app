Fyll ut skjemaet og bli Trumf-medlem!

Som Trumf-medlem sparer du bonus på din medlemskonto når du handler hos Trumf sine partnere. Du får tilgang til din kjøpshistorikk og du kan administrere din profil og dine samtykker ved å logge deg inn på trumf.no.

Bekreft vilkår og bli Trumf-medlem!

Ved å bli Trumf-medlem aksepterer du følgende:

  • Trumf kan bruke opplysningene om deg og dine kjøp (eksempelvis om varer, tjenester og bruk av betalingskort knyttet til Trumf) til å beregne bonus og administrere medlemskapet. Dersom du knytter medlemskapet ditt til flere medlemmer i et delt medlemskap, vil opplysninger om deg og dine kjøp deles med disse medlemmene.
  • Trumf kan sammenstille dine personopplysninger med andre virksomheters personopplysninger for å oppnå mest mulig korrekte kundedata. Et typisk eksempel er adresseoppdateringer.
  • Trumf og Trumfs partnere kan kontakte deg via SMS, e-post, vanlig post eller andre alternativer ut i fra valgene på min profil som innlogget på trumf.no.
  • Trumf anonymiserer opplysninger om deg og dine kjøp, og kan bruke dem til formål som ikke kan knyttes til deg personlig.
  • Øvrige medlemsvilkår.
